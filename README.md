📌 MUVV Rotas 2

Uma aplicação frontend em TypeScript + Vite para demonstração e gerenciamento de rotas.
Desenvolvida como projeto prático para organização de caminhos e navegação.

🧠 Sobre o Projeto

Esse projeto foi construído com o objetivo de:

Demonstrar rotas e navegação entre componentes ou páginas;

Servir como base leve e moderna para novos projetos;

Trabalhar com TypeScript, Vite e estrutura de componentes organizada.

💻 Como Rodar o Projeto

Siga os passos abaixo para iniciar a aplicação localmente:

1. Clone o repositório
git clone https://github.com/Victor30011991/muvv_rotas_2.git
2. Entre na pasta
cd muvv_rotas_2
3. Instale as dependências
npm install
# ou
yarn
4. Inicie o servidor de desenvolvimento
npm run dev
# ou
yarn dev

Em seguida, abra no navegador o endereço:

http://localhost:5173
🛠 Desenvolvimento

Se você quiser compilar e gerar produção:

npm run build
# ou
yarn build

Para testar o build:

npm run preview
# ou
yarn preview
